# Third-party Notices

This repository must keep third-party attribution and notice obligations
separate from the Zayd source-code license and separate from any dataset rights.

## When to add an entry

Add an entry before merging any third-party code, asset bundle, generated
artifact with embedded third-party code, or documentation material that carries
its own notice obligations.

Human license review is required before merge.

## Entry Template

Use one section per component:

```text
## Component: <name>
- Type: <library | framework | snippet | asset | documentation>
- Upstream project: <name>
- Source repository: <url>
- Source version/commit: <version or commit hash>
- Original license: <license identifier>
- Local path(s): <path list>
- Copyright notice: <required notice text or reference>
- Modification status: <unmodified | modified>
- Modification summary: <short summary>
- NOTICE requirements: <yes/no + details>
- Review status: <pending | approved | rejected>
- Reviewed by: <name or role>
- Review date: <YYYY-MM-DD>
```
