# Licenses Directory

Use this directory for supplemental license texts, third-party license copies,
or dataset license references that must ship with the repository.

## Intended Contents

- full license texts for bundled third-party dependencies when required
- custom permission letters or archived public license references when allowed
- dataset-specific license copies or references for redistributable sample data

## Rules

- Do not place private licensed corpora, religious texts, or restricted
  datasets here.
- Do not assume a source-code license applies to datasets.
- Add matching entries to `THIRD_PARTY_NOTICES.md` and `CODE_PROVENANCE.md`
  when the directory is used for third-party code.
- Keep unknown or unreviewed data rights out of public releases.
