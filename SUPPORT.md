# Support

## Scope

Support covers repository setup, documentation questions, issue triage, and clarification of project policy.

## Not Covered

- Emergency response
- Legal advice
- Religious rulings
- Production incident handling with guaranteed response times
- Private account recovery

## How to Ask for Help

1. Read [`README.md`](README.md) and [`CONTRIBUTING.md`](CONTRIBUTING.md).
2. Search the existing documentation in [`docs/`](docs/README.md).
3. Open a public issue only for non-sensitive project questions.
4. Use the private security process in [`SECURITY.md`](SECURITY.md) for security concerns.

## Privacy

Do not include private user conversations, secrets, passwords, personal data, or restricted datasets in support requests.
